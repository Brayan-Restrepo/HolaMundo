Esta Biblioteca es una adaptación realizada como material de trabajo para varios cursos de Ingeniería de Software, para el modelado e implementación de soluciones de software, con una orientación a objetos y una arquitectura en tres capas. La arquitectura se implementa con Java EE, utilizando JSF, EJB y JPA para las capas de presentación, negocio y persistencia respectivamente. El caso de estudio original es presentado por Stephen Schach, en el libro Object Oriented and Classical Software Engineering, New York: McGrawHill, 2010, Pag. 261.
